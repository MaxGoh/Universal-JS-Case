# Universal-JS-Case

==========

## Getting Started

### Installation

`npm install javascript-universal-case`

### Testing

`npm test`

## Usage


```
import universalCase from 'javascript-universal-case';

universalCase.toCamelCase("snake_case") // snakeCase
universalCase.toSnakeCase("snakeCase") // snake_case
```
